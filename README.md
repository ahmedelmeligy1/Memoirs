# 🌟 ذاكرة الأجيال — Memory Lane

صفحة شخصية لحفظ ذكريات الأبناء عبر المراحل العمرية.

## 🚀 رفع على Vercel

### الطريقة الأولى — عبر الموقع (الأسهل):
1. اذهب إلى [vercel.com](https://vercel.com) وسجّل دخول
2. اضغط **Add New Project**
3. اضغط **Upload** واختر هذا الفولدر كاملاً
4. اضغط **Deploy** — خلاص! 🎉

### الطريقة الثانية — عبر GitHub:
1. ارفع الفولدر على GitHub كـ repository جديد
2. في Vercel اضغط **Import Git Repository**
3. اختر الـ repo واضغط **Deploy**
